# Dj_Convertisseur Project

## 📖 Description
**Dj_Convertisseur** est une application web développée initialement en Java EE puis migrée vers Spring Boot.  
Elle permet de convertir facilement des documents entre plusieurs formats :
- 📄 **Word → PDF**
- 📄 **PDF → Word**
- 📊 **PDF → Excel**

Après la conversion, l’utilisateur peut **télécharger le fichier** et le partager directement via **WhatsApp** ou **Telegram**.

Ce projet vise à offrir un outil simple et pratique pour les étudiants, professionnels et entreprises qui manipulent régulièrement des documents.

##  Technologies
- Java EE (version initiale)
- Spring Boot (version actuelle)
- Apache POI (Word/Excel)
- PDFBox / iText (PDF)
- Render (déploiement)
